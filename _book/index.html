书籍简介

本书系统介绍 Stata MCP 的实战指南，涵盖从理论基础到实操落地的全流程：

  • 核心内容：包括 Stata MCP 安装配置、AI 助手集成、高级功能应用等
  • 技术适配：VS Code、Cursor、Antigravity 等 IDE 环境配置
  • 案例导向：通过 14+ 个学术应用场景演示 AI 赋能 Stata 研究，附完整 Prompt 示例
  • 资源配套：提供配置选项速查、快捷键手册、常见错误排查指南等实用附录

本书适用于计量经济学研究者、实证分析师、高校师生等群体，既可作为 Stata MCP 入门教程，也可作为进阶研究的参考手册。

如何使用本书

  • 导航方式：左侧侧边栏可浏览章节结构（默认折叠一级章节，点击展开），支持关键词搜索（输入框位于侧边栏顶部）
  • 代码交互：书中代码块可直接复制运行（需配置对应 Stata 环境）
  • 资源获取：所有数据、代码及扩展资料均托管于 GitHub 仓库（见侧边栏底部链接）
  • 版本更新：本书将持续迭代，最新版本及更新日志可通过主页查询
